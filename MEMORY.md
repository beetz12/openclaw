# Memory

## Business Context

Software consultancy providing custom development and technical partnerships.

## Common Terms

| Term                 | Meaning                                                     |
| -------------------- | ----------------------------------------------------------- |
| **Kickoff call**     | Initial client alignment meeting (goals, timeline, metrics) |
| **Point of contact** | Primary client relationship manager                         |
| **Project portal**   | Client-facing progress tracking system                      |

→ Full glossary: memory/glossary.md

## Client Onboarding Flow

1. Contract signed
2. **Welcome email** sent (introduces team, next steps, portal access)
3. **Kickoff call** scheduled (within few days)
4. **Portal access** provided
5. Project begins

## Communication Patterns

- Professional, action-oriented tone
- Clear next steps in all communications
- Provide contact info for urgent needs
- Set expectations early (timelines, touchpoints)
